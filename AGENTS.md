# GitHobbit — Developer Tools with Fantasy Flair

## What This Is

A fantasy-themed parody of developer tooling culture. The site is the
marketing. The tools (Treebeard, Gandalf) are the product.

Domain: githobbit.com

## Current State

Static landing page at githobbit.com. Next phases: merch store, then
GitHub App bots (Treebeard for PR hygiene, Gandalf for CI guardian).

## Guidelines

- Use the `daily` or `creative` agent depending on the task.
- The fantasy theme is the differentiator — voice should be consistent,
  useful first, funny second.
- Keep the product separable from the theme at all times.

## Voice

- Consistent enough to feel like a character, not random jokes
- Useful first — every comment should contain actionable information
- Funny second — the comedy is delivery, not content
- Configurable — teams can adjust tone from "full character" to "subtle"
